Module 2: Neural Networks & Deep Learning

Neural Networks are a subset of Machine Learning inspired by the biological structure of the human brain. While traditional algorithms often rely on manual feature engineering, Neural Networks excel at learning hierarchical representations directly from raw data.

The Fundamental Building Block: The Perceptron

At its core, a neural network is composed of many artificial neurons (or nodes) called perceptrons. Each perceptron takes multiple inputs, multiplies them by weights ($w$), adds a bias ($b$), and passes the result through an activation function to determine the output[cite: 5].

Network Architecture

A Neural Network is organized into layers[cite: 5, 6]:

  • Input Layer: Receives the raw data (e.g., pixels of an image)[cite: 5, 6].
  • Hidden Layers: Intermediate layers where the actual "learning" happens. Deep Learning refers to networks with many hidden layers[cite: 5, 6].
  • Output Layer: Produces the final prediction or decision[cite: 5, 6].

How the Network Learns

Learning is an iterative process that relies on two primary mechanisms[cite: 5, 6]:

  • Forward Propagation: Data passes through the network, layer by layer, to generate an output which is compared to the actual target (the "ground truth")[cite: 5, 6].
  • Loss Function: A mathematical calculation of the error—the difference between the model's prediction and the actual value[cite: 5, 6].
  • Backpropagation & Optimization: The network works backward to calculate how much each weight contributed to the error. An optimizer (like Gradient Descent) then adjusts the weights to minimize that error[cite: 5, 6].

Key Components for Success

  • Activation Functions: Functions like ReLU, Sigmoid, or Softmax that introduce "non-linearity," allowing the network to learn complex patterns[cite: 5, 6].
  • Weights and Biases: The parameters the network adjusts during training to improve accuracy[cite: 5, 6].
  • Deep Learning: Using networks with many layers to solve complex tasks like object detection and speech recognition[cite: 5, 6].

Why use Neural Networks?

  • High-Dimensional Data: Effective at handling thousands of input features simultaneously[cite: 5, 6].
  • Feature Discovery: Automatically discovering patterns that humans might not even be able to define[cite: 5, 6].
  • Scalability: Performance typically continues to improve as you add more data and more layers[cite: 5, 6].
